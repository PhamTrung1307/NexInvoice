# NexInvoice Frontend

React + Vite frontend for NexInvoice, built as a Vietnamese enterprise dashboard for customer, project, contract, invoice, payment, notification, report, and settings workflows.

Author: Pham Nguyen Anh Trung
Display name: Trung

## Scripts

```powershell
npm install
npm run dev
npm run build
```

## Notes

- UI text is Vietnamese.
- Code identifiers remain English for maintainability.
- API calls are centralized in `src/api`.
- Authentication tokens are attached through the shared Axios instance.
